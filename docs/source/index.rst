Welcome to ExpiryEats' documentation!
=====================================

**ExpiryEats** is an application designed to assist individuals with food management and finding new recipes.
It pulls data from `The Meal Database <https://https://www.themealdb.com/api.php/>`_
and offers a *simple* and *intuitive* way to manage your inventory.

.. note::

   This project is under active development.
